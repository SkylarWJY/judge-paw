# Miro Board — Judge Paw (From Board to Build)

The public Miro board, laid out as a **courtroom**. Copy each frame's sticky notes
directly. Suggested left→right reading order. Use the Mirotone palette (pinks/warm
neutrals) to match the Judge Paw theme.

---

## Frame 1 — ⚖️ The Problem (opening statement)
- Every couple has the same unsolved workflow: **"who's actually right?"**
- Settled today by: sulking · guessing · asking the group chat
- There is **no neutral third party** in a relationship
- Result: the fight stalls, nobody feels heard, it repeats next week
- *Stalled workflow we're here to unblock* ← tie to hackathon theme

## Frame 2 — 🐾 The Solution
- **Judge Paw** — the first AI relationship court
- Hears **both sides**, renders a fair, warm verdict
- Output = **Ruling** + **Reasoning** + **Sentence** (a make-up action)
- Persona: a beloved animal judge — fair, warm, never cruel

## Frame 3 — 🚶 User Journey (sticky notes as a left→right flow with arrows)
1. Couple has a fight 😤
2. Opens Judge Paw
3. Each partner types their side
4. Judge Paw deliberates 🐾
5. Verdict delivered: Ruling / Reasoning / Sentence
6. They make up 💕 (case closed)

## Frame 4 — 🏛️ The Courtroom Flow (the engine, as a diagram)
- INPUT: topic + Partner A statement + Partner B statement
- → Judge Paw weighs both (must cite a valid point from EACH)
- → Safety check: abusive input → declines gently
- → OUTPUT: Ruling (proportional, not binary) · Reasoning · Sentence
- Built with: **Claude** (structured verdict) · **Kiro** (spec-driven build)

## Frame 5 — 📊 Metrics (how we know it works)
- **Resolution rate** — % of cases that end in "case closed / made up"
- **"Felt heard" score** — both partners rate if the verdict understood them
- **Repeat-fight rate** — does the same dispute come back?
- North star: *both partners feel the verdict was fair*

## Frame 6 — 🛠️ Board → Build (the hackathon story)
- Planned here in **Miro** (this board)
- Built with **Kiro** — spec-driven (`requirements → design → tasks`)
- Shipped: working Judge Paw prototype + 3-min demo
- One line: **Board → Build → Verdict**

## Frame 7 — 🚀 Pitch / Vision (closing)
- TAM: every couple, everywhere, forever
- Wedge: the recurring "who's right" fight
- Next: shareable verdicts · history · couples' track record
- Tagline: *"Let the paw decide."* 🐾⚖️
